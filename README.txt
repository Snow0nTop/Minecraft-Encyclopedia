# Minecraft Encyclopedia

Courte description (1–2 phrases).

	Te voici dans la bibliothèque de Minecraft !
    Ceci est un petit aperçu des différents blocs, minerais, mobs, enchantements et potions présentes dans le jeu.
    Amuse-toi bien !

## Comment tester
- Ouvrir `index.html` dans un navigateur. Mais avant, il faut ouvrir le dossier complet du projet dans VSCode, et utiliser LiveServer.

## Crédits
### Sons
- Nous avons pris l'ambiance du jeu, qui est dans nos vidéos directement.

### Images
- Toutes les images ont été prises ou faites sur Photoshop par nous-même, ce sont des screenshots de Minecraft.

## Remarques
- Il est recommandé de jouer en plein écran, sinon quelques bugs de placement des zones cliquables (rectangles) auront lieu.
